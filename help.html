How can you help win the race against climate change?

We understand that you can't stop using plastic immediatly. That's why we created 3 simple guidelines. Preferably only use bio-plastics. These don't have as big as an impact as normal plastic. Next up recycle as much plastic as you can, by sorting plastic. This way the plastic that you use get a second home. And last but not least, don't throw away any bio-plastics outside. Try to always find a trashcan.
